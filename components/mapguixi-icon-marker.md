# MapGuixiIconMarker 桂溪地图图标标记

## Props

| prop     | type                 | desc               | default   |
| -------- | -------------------- | ------------------ | --------- |
| color    | `string`             | 图标颜色           | `require` |
| content  | `Enum(string, node)` | 弹出内容           | `require` |
| name     | `string`             | 图标名称           | `require` |
| isActive | `boolean`            | 图标是否为活跃状态 | `false`   |

## Events

| event   | args | desc                 |
| ------- | ---- | -------------------- |
| onClick | `()` | 标记被点击的回调函数 |
